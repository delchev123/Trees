# BST и AVL дърво – C#

## Описание
Проектът реализира двоично наредено дърво (BST) и AVL дърво.
Поддържа добавяне, търсене и обхождане на елементи.

## Допълнително
AVL дървото се самобалансира чрез ротации (LL, RR, LR, RL),
което гарантира логаритмична сложност на операциите.
